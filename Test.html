Quick deploy commands

Copy this file to index.html and use any static host.

# GitHub Pages (one-time)
git init
git add index.html
git commit -m "add deployment test"
git branch -M main
git remote add origin <your-repo-url>
git push -u origin main

# Deploy with Vercel
vercel deploy --prod

# Netlify (drag & drop or CLI)
netlify deploy --prod

Tip: For GitHub Pages ensure your repo has a branch named gh-pages or use the Pages settings.
